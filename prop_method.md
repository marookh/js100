### Solution

```JavaScript
let trees = ['birch', 'pine', 'sequoia', 'palm tree'];

trees[trees.length - 1];  // Returns: 'palm tree' as the last element.
trees.pop();              // Removes the last element of the array and returns removed one which is 'palm tree' in this case.
trees[trees.length - 1];  // Returns: 'sequoia' as the last element.
